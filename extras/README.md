# Extras

Extra utilities, documentation, and resources related to the GPStar project.

## Installation Guides

* Attenuator Install Guide.pdf - Installation for the Frutto Technologies Attenuator electronics kit.
* GPStar 3D Printed Neutrona Wand Installation Instructions.pdf - Installation guide for 3D Printed Neutrona Wands.
* GPStar Mack's Factory Neutrona Wand Installation Instructions.pdf - Installation guide for the Mack's Factory Neutrona Wand.
* GPStar Hasbro Proton Pack Kit Installation Manual V4.pdf - Installation guide for the Hasbro Proton Pack and Neutrona Wand.
* GPStar PSTT Assembly Instructions.pdf - Assembly guide for the GPStar Proton Stream Target Trainer.

## Operation Guides

* GPStar I Operational Guide.pdf - Operation guide for the original GPStar Proton Pack and Neutrona Wand.
* GPStar II Operational Guide.pdf - Operation guide for the new GPStar II Proton Pack and Neutrona Wand.
* GPStar PSTT Operation Guide.pdf - Operation guide for the GPStar Proton Stream Target Trainer.

## AI Integration

* rag (directory) - Contains a vector index database and text chunks required for Retrieval-Augmented Generation (RAG)
